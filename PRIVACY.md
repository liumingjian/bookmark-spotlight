# Privacy Policy

Bookmark Spotlight does not collect, store, transmit, sell, or share user data.

The extension requests the Chrome `bookmarks` permission only to read the user's
local Chrome bookmarks when the search overlay is opened. Bookmark titles and
URLs are used locally in the browser to show search results.

The extension uses host access only to display the bookmark search overlay on
ordinary web pages after the user triggers the keyboard shortcut. It does not
read, collect, or transmit page content.

No remote code is used. All extension code is included in the extension package.
